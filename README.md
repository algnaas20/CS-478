# CS-478 Review

An interactive web-based review page for **CS-478 Computer Graphics**.

## Live Demo

[https://algnaas20.github.io/CS-478/](https://algnaas20.github.io/CS-478/)

## Overview

This project provides a standalone HTML review page designed to help students study for the CS-478 exam. It combines review questions, quiz-style practice, Arabic question translations, and clear answer explanations in both Arabic and English.

The page is mobile-friendly and can be accessed easily through GitHub Pages or opened locally in any browser.

## Contents

- 174 multiple-choice questions
- Midterm coverage from Modules 1-5
- Final coverage from Modules 6-12
- Module 13 removed based on exam scope
- No essay questions included

## Features

- Clean standalone HTML review page
- Mobile-friendly layout for quick studying
- Arabic translation under every question
- Arabic explanation for every answer
- Full English explanation below each Arabic explanation
- Explanations describe why the correct answer is correct and why the other options are not
- Wrong answers automatically reveal the explanation
- Review Wrong Only mode for retrying missed questions
- Local browser saving for selected answers
- Easy access through GitHub Pages

## Purpose

The goal of this review page is to make exam preparation easier by keeping the important questions, translations, and explanations in one place. The explanations are written to be clear and self-contained, so students can understand the main ideas without constantly going back to the course material.
